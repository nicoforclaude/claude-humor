## Installation

### Local Testing

1. Clone this repository
2. Add the marketplace:
   ```
   /plugin marketplace add C:\KolyaRepositories\nicoforclaude\claude-humor
   ```
3. Install plugins:
   ```
   /plugin install joke-teller@claude-humor
   /plugin install roaster@claude-humor
   ```

### From GitHub (coming soon)

Once published, you'll be able to install directly:
```
/plugin marketplace add https://github.com/nicoforclaude/claude-humor
```
